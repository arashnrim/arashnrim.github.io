Here's the history.

Some time in 2020, I had an idea to embark on discovering new technologies given the free time that I had. As a result of this, the Sandbox was born. The main idea I had for the Sandbox was to host many experimental projects in different branches to ensure that each project is somewhat isolated from one another, yet navigation remains easy.

Thus far, there are two projects as part of the Sandbox family of experimental projects: pyweather, which eventually matured beyond an experimental state (though still related to the Sandbox), and pyscraper.

To document my learnings and findings, each project uniquely contains a learning log that is shared in the project README document. I hope that, through these findings, it may help anyone else on their journey to discovering new technologies or to remind me in the future of what has been learnt.
