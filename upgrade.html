Here's the history.

Upgrade was a group project I created for a school project, Project Decimus. The goal of the project was to create an iOS app that served as a conclusion to the school curriculum covering design, innovation, and creation.

The project took place in 2019 over the course of almost three-quarters of the year. With the help of consultations and regular checkups, we were ensured that our group project was as good a quality as possible with advice from the teachers.

We went through many processes of thinking, and the final app idea definitely was not our first. Nonetheless, with close collaboration, we managed to settle on an idea and work closely to design, develop, and plan the app. The project concluded with a group presentation and showcase.

Here come the technicalities.

This project utilises a cool technology, and I utilised it the best I could to what I knew to make the end result the best effort. This app uses the following tech stack listed below.


Swift

I created this app using the Apple-created programming language of Swift and the Xcode IDE. I created the app using the Storyboards variant of an Xcode project, using a mix of visual UI drag-and-drop work with a lot of manual programming. In addition, I used the Cocoapods dependency manager to manage external libraries required.


Firebase

For authentication and data storage, I used the Firebase platform to handle these areas relatively easily. Its simple-to-understand documentation alongside tutorial videos accelerated my learning of how to use it. Initially starting out with Realtime Database, my team made the decision to use Cloud Firestore instead. We also used Authentication for, well, authentication.
